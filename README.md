# ⚡ EMP Vault Survival Guide

## Overview
This project provides a **real-world, physics-based approach** to protecting digital storage (SD cards / USB) from electromagnetic pulse (EMP) events.

> ❌ There is NO such thing as an EMP-proof storage device  
> ✅ Protection comes from **shielding + isolation + redundancy**

---

## 🔥 Core Concept

An EMP destroys electronics by inducing voltage in circuits.

The solution:
- Build a **Faraday enclosure**
- Prevent energy from reaching the storage
- Use **multiple redundant copies**

---

## 🧱 Architecture

### Base Stack
```
SD Card
→ Plastic Sleeve
→ Foam Insulation
→ Faraday Bag
→ Threaded Steel Capsule
```

### Overkill Stack
```
SD Cards (multiple copies)
→ Insulated pack
→ Faraday pouch
→ Steel capsule
→ Outer metal container
→ Stored in separate locations
```

---

## 🛠️ DIY Build Guide

### 1. Outer Shell
- Steel pipe + threaded caps
- MUST be conductive
- No gaps

### 2. Seam Preparation
- Sand threads
- Remove paint / coatings
- Ensure metal-to-metal contact

### 3. Internal Insulation
- Foam / rubber / plastic
- Storage must NOT touch metal

### 4. Layering
- Add Faraday bag inside capsule
- Optional second outer container

---

## ⚠️ Common Failures

- Loose lids
- Painted threads
- Rubber seals
- Storage touching metal
- Single device (no redundancy)

## 🔌 USB Adapter Rule

If using SD → USB adapter:

✔ Store adapter separately inside cage  
❌ NEVER leave it plugged in  
❌ Avoid exposed conductive paths  

---

## 🛡️ Advanced Notes

- Do NOT ground the container (floating cage is better)
- Use desiccants for long-term storage
- Keep storage centered inside enclosure
- Avoid any external conductive attachments

---
